 Thessaloniki04 Customization
************************************************************************

 Thessaloniki04 provides simplified Google Maps 
 support for requests and offers.
 
 In addition, Thessaloniki03 provides suppport for 
 specifying a timespan for which only hours
 can be specified.
 So this is extremely simple, but also very inflexible
 and may not be well suited for many business cases.
