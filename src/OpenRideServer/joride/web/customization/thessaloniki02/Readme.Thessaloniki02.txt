************************************************************************
 Thessaloniki02 Customization
************************************************************************

 Thessaloniki02 provides simplified Google Maps 
 support for requests and offers.
 As does Thessaloniki01 extension.
 
 In addition, Thessaloniki02 provides suppport for 
 specifying a timespan for which days/hours/minutes
 can be specified seperately.
 Also, there is JS in place for normalizing the input,
 such that hours are in 0...23 and minutes are in 0...59
 while preserving the overall time.
